Python Numbers

Python Numbers

The three numeric types in Python:

  • int
  • float
  • complex

Numeric types are instantiated when you assign a value to them:

>>> x = 30 # int
y = 31.2 # float
z = 1j # complex

A complex number consists of a real part and an imaginary part, with the imaginary part denoted by a number followed by the imaginary unit. For instance, in 4j+3, 4j is the imaginary part where j represents the imaginary unit, and 3 is the real part.

Example
print ( type(x) )
print ( type(y) )
print ( type(z) )
Int

An integer (int) is a whole number, either positive or negative, without decimal points, and can be of unlimited length.

Example
Integer:
x = 1
y = 327947392
z = -438982

print ( type(x) )
print ( type(y) )
print ( type(z) )

Float

A float, or "floating-point number," is a number that can be positive or negative and may contain one or more decimal points.

Example
Floats:
x = 21.10
y = 31.0
z = -3.59

print ( type(x) )
print ( type(y) )
print ( type(z) )

Floats can also represent scientific numbers using "e" to denote the power of 10.

Example
Flaots:
x = 5e3
y = 2je6
z = -98.5e232

print ( type(x) )
print ( type(y) )
print ( type(z) )

Complex

Complex numbers are written with a "j" as the imaginary part:

Example
Complex:
x = 3 +5j
y = 5j
z = -5j

print ( type(x) )
print ( type(y) )
print ( type(z) )

Type Conversion

You can convert between different types using the int(), float(), and complex() methods:

Example
Convert from one type to another:
x = 3      # int
y = 2.8   # float
z = -5j   # complex

#convert from int to float:
a = float(x)

#convert from float to int:
b = int(y)

#convert from int to complex:
c = complex(x)

print(a)(x)
print(b)(x)
print(c)(x)

print (type(a))

Note: Complex numbers cannot be converted into other numerical types.


Random Number

Python doesn't feature a random() function for generating random numbers directly. However, Python offers a built-in module named random for this purpose:

Example
Import the random module and output a random number between 1 and 9:
import random
print (random.randrange( 1 , 3))
