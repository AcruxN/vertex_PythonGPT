Python Conditions and If statements

There are six type of common logical conditions from mathematics:

  • Equals: a == b
  • Not Equals: a != b
  • Less than: a < b
  • Less than or equal to: a <= b
  • Greater than: a > b
  • Greater than or equal to: a >= b

These conditions most commonly in "if statements" and loops.


An "if statement" is written by using the if keyword.


Example
If statement:
a = 34
b = 102
if b > a:
     print("b is greater than a")

In this example, we use two variables, a and b, to test if b is greater than a. Since a is 33 and b is 200, we print "b is greater than a" because 200 is indeed greater than 33.

Elif

The elif keyword to set another condition if previous condition is no true.


If commonly use in "if statements" and loops.


Example
a = 44
b = 44
if b > a:
     print("b is greater than a")
elif a == b:
     print("a and b are equal")


In this example , the first condition is not true, but the second condition, theelif block is true, so it will print that "a and b are equal".

Else

If every condition if false, the else keyword will function



Example
a = 300
b = 44
if b > a:
     print("b is greater than a")
elif a == b:
     print("a and b are equal")
else :
     print("a is greater than b")

And

The and keyword is a logical operator, it combine the condition and return true if both of them are correct:


Example
Test if a is greater than b, AND if c is greater than a:
a = 300
b = 22
c = 400
if a > b and c > a:
    print ( "Both conditions are True")

Or

The and keyword is a logical operator, and return False while two of the condition is false, otherwise it will return True:


Example
Test if a is greater than b, OR if c is greater than a:
a = 300
b = 22
c = 400
if a > b or c > a:
    print ( "At least one of the conditions is True")

Not

The not keyword is a logical operator, that used to reverse the result:


Example
Test if a is NOT greater than b:
a = 300
b = 22
if not a > b:
    print ( "a is NOT greater than b")

Nested If

You can have if statements inside if statements, this is called nested if statements.


Example

if x > 10:
    print ( "Above ten, ")
    if x > 10:
        print ( "and also above 20! ")
    else
        print ( "but not above 20.")

The pass Statement

if statements cannot be left empty. If you need to have an if statement with no code inside, use the pass statement to prevent errors.


Example
a = 300
b = 44
if b > a:
    pass
