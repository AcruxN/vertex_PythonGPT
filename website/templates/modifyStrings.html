Python - Modify Strings

Python provides a collection of built-in methods specifically designed for string manipulation.
Upper Case

Example
The upper() function converts the string to uppercase:
a = "Hello, World!"
print(a.upper())
Lower Case

Example
The lower() function converts the string to lowercase:
a = "Hello, World!"
print(a.lower())
Remove Whitespace

Whitespace refers to the space before and/or after the actual text, which you often need to eliminate.
Example
The strip() method eliminates any whitespace from the start or end:
a = "Hello, World!"
print(a.strip()) # returns "Hello, World!"
Replace String

Example
The replace() method substitutes one string with another:
a = "Hello, World!"
print(a.replace( "H", "J" ))
Split String

The split() method generates a list where the text separated by the specified separator forms the list items.
Example
The split() method divides the string into substrings whenever it encounters occurrences of the separator:
a = "Hello, World!"
print(a.split( ",")) # returns ['Hello', 'World!']
