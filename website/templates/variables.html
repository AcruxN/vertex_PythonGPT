Python Variables

Variables

Variables serve as containers for storing data values.

Creating Variables

Python lacks a specific command for declaring a variable.

A variable is instantiated the moment you first assign a value to it.

Example
x = 5
y = "John"
print(x)
print(y)

Variables don't require declaration with any specific type and can even change type after their initialization.

Example
x = 4
y = "Sally"
print(x)
# is now of type str
Casting

To specify a variable's data type, casting can be utilized.

Example
x = str(3)
y = int(3)
z = float(3)
Get the Type

To determine the data type of a variable, you can employ the type() function.

Example
x = 5
y = "John"
print(type(x))
print(type(y))
