Python Comments

Comments serve to elucidate Python code.

Comments enhance code readability.

Comments can also halt execution during code testing.

Creating a Comment

Python supports commenting for in-code documentation.

Comments in Python begin with a #, and the rest of the line is considered as a comment.

Example
Comments in Python:
#This is a comment.
print ("Yo, Dude!")

A comment doesn't necessarily have to explain the code; it can also prevent Python from executing certain lines:

Example
Comments in Python:
#print("Yo, Dude!")
print ("Yo!,Dude!")
Multiline Comments

Python lacks a specific syntax for multiline comments.

To create a multiline comment, you can prefix each line with a #:

Example
Comments in Python:
#This is a comment
#written in
#more than just one line
print ("Yo, Dude!")

Alternatively, though not the intended use, you can employ a multiline string.

Given that Python disregards string literals not assigned to a variable, you can incorporate a multiline string (enclosed in triple quotes) in your code and embed your comment within it:

Example
Comments in Python:
#print("Yo, Dude!")
print ("Yo!, Dude!")
