A lambda function is a small anonymous function defined with the 'lambda' keyword.

A lambda function can have only one expression, but can can have any number of arguments. It normally restricted to a single line



Syntax
lambda arguments : expression

The expression is executed and the result is returned:

Example
For example, when the lambda expression is defined to add 30 to a, after running below program and print the result, it will be 35
x = lambda a : a + 30
print(x(5))

Lambda functions can take many arguments:

Example
Multiple argument c with argument d will return the result 12 after assign 3 to c and 4 to b:
x = lambda c, d : c * d
print(x( 3, 4))
The Importance of Lambda Functions?

The true strength of lambda functions becomes evident when they are used as anonymous functions within another function.

For example, consider a function that takes one argument and multiplies it by an unspecified number:

def myfunc(n):
      return lambda c : c * n

The above function can make a function that always time with the number you set

Example
def myfunc(n):
      return lambda c : c * n

mytripler = myfunc(3)

print(mytripler(0))

You can replace the n with any number to make it multiple with any number



Or, you can make two or more function in same program


Example
def myfunc(n):
      return lambda a : a * n

mydoubler = myfunc(2)
mytripler = myfunc(3)

print(mydoubler(33))
print(mytripler(33))
