Python Strings

Strings

In Python, strings can be enclosed in either single or double quotation marks.

'Yo!' is the same as "Yo!".

To output a string literal, you can use the print() function:

Example
print ("Yo!")
print ('Yo!')
Quotes Inside Quotes

You can include quotes within a string as long as they don't match the surrounding quotes:
Example
print ("It's alright")
print ("I am 'Ian'")
print ('I am "Ian"')
Assign String to a Variable

To assign a string to a variable, you use the variable name followed by an equal sign and the string:
Example
a = "Yo!"
print(a)
Multiline Strings

To assign a multiline string to a variable, you can use three quotes:
Example
You can use three double quotes:
a = """Example string line one,
followed by line two,
and then line three.
print(a)
Or three single quotes:
Example
You can use three double quotes:
a = '''Example string line one,
followed by line two,
and then line three.'''
print(a)
