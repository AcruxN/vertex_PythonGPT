Python - Slicing Strings

Slicing

You can retrieve a range of characters using the slice syntax.

To extract a portion of the string, specify the starting index and the ending index separated by a colon.

Example
Retrieve the characters from position 2 up to position 5 (exclusive):
b = "Hello, World!"
print(b[2:5])
Note: Indexing in Python starts from 0.
Slice From the Start

If you leave out the start index, the range will start at the first character:

Example
Retrieve the characters from the beginning to position 5 (not included):
b = "Hello, World!"
print(b[:4])

Slice To the End

If you omit the end index, the range will extend to the end:

Example
Retrieve the characters starting from position 2 until the end:
b = "Hello, World!"
print(b[1:])
Negative Indexing

Use negative indices to initiate the slice from the end of the string:

Example
Retrieve the characters:
From: "o" in "World!" (position -5)
To, excluding: "d" in "World!" (position -2)
b = "Hello, World!"
print(b[-5:-2])
