Python Functions

A function is a reusable block of code designed to perform a specific task.
It takes input, processes it, and returns an output.


Creating a Function

In Python you can use def keyword to create a function.

Example
def first_function():
    print("Hellow word" )
Calling a Function

Example

Parameters or Arguments?

Parameters and arguments are information that are passed into a function.


From a function's perspective:


Parameter: the variable listed inside the parentheses in the function definition.


Argument: the value that is sent to the function when it is called.

Number of Arguments

By default, functions must be called with the correct number of arguments. This means that if your function needs two arguments, you must call the function with two arguments, no more and no less.


Example

Arbitrary Arguments, *args

f you're unsure about the number of arguments that will be passed to your function, prefix the parameter name with an asterisk (*) in the function definition.


This allows the function to accept a tuple of arguments and access each item as needed:


Example
If you don't know the number of arguments, add a * before the parameter name:

   print ("The largest things is" + things[2])

first_fucntion("Apple", "Watermelon", "Lemon" )

Arbitrary Arguments can be write as *args in Python documentations.

Positional-Only Arguments

You can specify that a function can have ONLY positional argumnets, or ONLY keyword arguments.

To specify that a function can have only positional arguments, add , / after the arguments:


Example
def my_function(x):
      print ( x )

my_function( 3 )

Without the , / you are actually allowed to use keyword arguments even if the function expects positional arguments:


Example
def my_function(x):
      print ( x )

my_function( x = 3 )

But when adding the , / you will get an error if you try to send a keyword argument:

Example
def my_function(x, /):
      print ( x )

my_function( x = 3 )

Keyword-Only Arguments

To specify that a function can have only keyword arguments, add *, before the arguments:

Example
def my_function(*, x):
      print ( x )

my_function( x = 3 )

But when adding the *, / you will get an error if you try to send a positional argument:

Example
def my_function(*, x):
      print ( x )

my_function( x = 3 )

To specify that a function can have only keyword arguments, add *, before the arguments:

Example
def my_function(*, x):
      print ( x )

my_function( x = 3 )

Combine Positional-Only and Keyword-Only

Combine Positional-Only and Keyword-Only

Any argument before the / , are positional-only, and any argument after the *, are keyword-only.

Example
def my_function(a, b, /, *, c, d):
      print ( a + b + c + d )

my_function( 5, 6, c = 7, d = 8 )

Recursions

Python allows function recursion, meaning a function can call itself. This is useful in both math and programming for looping through data to produce results.


Developers should be cautious to avoid infinite loops and excessive resource use. When used properly, recursion is efficient and elegant.


For example, the tri_recursion() function calls itself with the variable kwhich decreases by 1 each time. The recursion stops when k reaches 0.


Recursion can be difficult to understand at first, but experimenting and modifying recursive functions can help.

Example
def tri_recursion(k):
if ((k > 0)):

      result = k + tri_recursion(k - 1)
      print(result)
else:

result = 0
return result

tri_recursion(6)
