String Concatenation

To join, or merge, two strings, you can employ the + operator.

Example
Merge variable a with variable b into variable c:
a = "Hello"
b = "Hello"
c = a + b
print(c)

Example
To include a space between them, append a " ":
a = "Hello"
b = "Hello"
c = a + " " + b
print(c)
