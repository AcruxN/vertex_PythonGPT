For Loops
for

A for loop iterates over a sequence, such as a list, tuple, dictionary, set, or string.

Using a for loop, you can execute a set of statements for each item in a sequence.


Example
Print each food in a food list:
food = ["nasi lemak", "durian", "abc"]
for x in food:
    print(x)

The for loop does not require an indexing variable to set beforehand.

Looping Through a String

For loop can iterate every characters including white space for string:

Example

It will print out H,E,L,L,O:

for x in "HELLO" :
    print(x)
The break Statement

The break statement can stop the looped while running

Example

Exit the loop when x is "banana". Below example will return apple only as the loop will break at banna

fruits = ["apple", "banana", "cherry"]
for x in fruits:
      print( x )
      if x == "banana" :
            break


The continue Statement

With the continue we can skip the current iteration and start next iteration:

Example

Do not print banana:

fruits = ["apple", "banana", "cherry"]
for x in fruits:
      if x == "banana" :
            continue
      print( x )


The range() Function

range() is use to determine the number of iteration of the loop

It will start from 0 and by default increments by 1 and end until the specified number.

Example

Using the range() function:

fruits = ["apple", "banana", "cherry"]
for x in range (5)
      print ( x )

Note that range(6) is not the values of 0 to 6, but the values of 0 to 5.

You can specify the starting value by adding a parameter. For example: range(3,7) specifies the values from 2 to 6

Example

Using the start parameter:

for x in range (2, 6):
      print ( x )

Besides, you can adjust increment value ofrange() by adding a third parameter: range(2, 30, 3)

Example

Using the increment parameter:

for x in range (2, 30, 3):
      print ( x )


Else in For Loop

The else keyword in a for loop specifies a block of code to be executed when the loop is finished.

Example

Show a message when the loop has ended

for x in range (3)
      print ( x )
else:
      print("finished!")

Note: The else block will NOT be executed if the loop is stopped by a break statement.

Example

The break block will stop the code running the else statment

for x in range (6) :
      if x == 3: break
      print ( x )
else:
      print("Finally finished!")


Nested Loops

A nested loop is a loop inside a loop.

The "inner loop"will be executed one time for each iteration of the "outer loop":


Example

Print each adjective for every fruit:

adj = ["red", "big", "tasty"]
fruits = ["apple", "banana", "cherry"]

for x in adj:
      for y in fruits:
            print(x, y)


The pass Statement

for function can be no content for specified reason if you add passkeyword.

Example
for x in [0, 1, 2] :
      pass
