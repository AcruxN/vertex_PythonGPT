Python - Output Variables

Output Variables

The Python print() function is commonly employed to display variables.

Example
x = "Python is fantastic"
print(x)

Multiple variables can be produced using the print() function, with each variable being separated by a comma.

Example
x = "Python"
y = "is"
z = "awesome"
print(x,y,z)

You can also utilize the + operator to display multiple variables.

Note: Please observe the space character after "Python " and "is "; without them, the result would be "Pythonisawesome".
For numbers, the + character works as a mathematical operator:
Example
x = 50
y = 100
print(x + y)

When attempting to combine a string and a number using the + operator within the print() function, Python will raise an error.

Example
x = 5
y = "Ian"
print(x + y)

The optimal method for outputting multiple variables in the print() function is to separate them with commas, a technique that even accommodates various data types.

Example
x = 50
y = "Ian"
print(x,y)
