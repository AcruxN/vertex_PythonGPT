Python Casting

Specify a Variable Type

There are occasions when you might need to specify a type for a variable. This can be achieved through casting. Python, being an object-oriented language, utilizes classes to define data types, including its primitive types.

Casting in Python involves the use of constructor functions:

  • int() - constructs an integer number from an integer literal, a float literal (by discarding all decimals), or a string literal (provided the string represents a whole number)
  • float() - constructs a floating-point number from an integer literal, a float literal, or a string literal (if the string represents a float or an integer)
  • str() - constructs a string from various data types, including strings, integer literals, and float literals
Example
Intergers:
x = int(2) # x will be 2
y = int(3.8) # y will be 3
z = int("4") # z will be 4
Example
Floats:
x = float(10) # x will be 10.0
y = float(20.8) # y will be 20.8
z = float("30") # z will be 30.0
w = float("40.2") # w will be 40.2
Example
Intergers:
x = str("L1") # x will be 'L1'
y = str(20) # y will be '20'
z = str(31.0) # z will be '31.0'
