Python Loops

Python has two kinds of loop :

  • while loops
  • for loops


The while Loop

ChatGPT Using the for loop, we can run a set of statements for each item in a list, tuple, set, or similar sequence.


Example
Print i if i lesser than 5:
i = 1
while i < 5:
      print ( i )
      i += 1

Note: Make sure to increment i, otherwise, the loop will run indefinitely. .

ChatGPT The while loop requires the necessary variables to be initialized. In this example, we need to define an indexing variable, i, and set it to 1.

The break Statement

The break statement can force the loop to stop even if the condition is true:

Example

Break the loop when i is 4:

i = 1
while i < 6:
      print( i )
      if i == 4:
            break
      i += 1
The continue Statement

Basically the continue statement will skip the current statement and run next iteration:

Example

Continue to the next iteration if i is 4:

i = 1
while i < 7:
      i += 4
      if i == 3:
            continue
      print( i )
The else Statement

The else statement allows us to execute a block of code once the condition is no longer true:

Example

Print a message once the condition is false:

i = 1
while i < 7:
      print( i )
      i += 1
else:
      print("i is bigger than 6 now")
