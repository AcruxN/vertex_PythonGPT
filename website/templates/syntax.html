Python Indentation

Indentation refers to the spaces at the start of a code line.

In many other programming languages, indentation is used solely for readability. However, in Python, indentation is crucial.

Python uses indentation to define blocks of code.

Example
if 5 > 2:
    print ("Five is greater than two!")

Python will give you an error if you skip the indentation:

Example
Syntax Error:
if 10 > 2:
print("Ten is greater than Two!")

The number of spaces you use is up to you as the programmer. While the most common practice is to use four spaces, you must use at least one.

Example
if 10 > 2:
    print("Ten is greater than two!")
if 10 > 2:
        print("Ten is greater than two!")

You must use the same number of spaces within the same block of code; otherwise, Python will produce an error:

Example
Syntax Error:
if 10 > 2:
    print("Ten is greater than two!")
     print("Ten is greater than two!")
